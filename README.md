# Framework Documentation Tool Specification

## Functionality:

- Generating documentation based on project structure and custom configuration.
- Generating custom [JSON based documentation format](custom-format.md).
- Plugin based architecture:
  - plugins for different languages
  - plugins for different frameworks
  
## API:

- --lang={lang name}, alias -l. Project language: js, ts, etc.
- --framework={framework name}, alias -fw. Project framework: angular, react, etc.
- --file={doc.json}, alias -f. Doc file generated by jsDoc, typeDoc, etc.
